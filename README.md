# Audio Wizard

For those who are tired of blindly changing parameters of audio effects in Godot.

> [!NOTE]
> Supported engine versions: 4.6+

![preview](readme_images/preview.png)

**Currently supported effects:**

- Amplify
- All filters
- Capture
- Chorus
- Compressor
- Delay
- Distortion
